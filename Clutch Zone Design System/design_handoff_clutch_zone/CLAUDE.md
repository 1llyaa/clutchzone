# CLUTCH ZONE — Design System (persistent project context)

Premium esport gaming club, České Budějovice, CZ. Tone: competitive, bold, professional — premium sports brand meets esport underground. Audience: CZ/SK gamers 16–35. Tagline: "Tvoje nová E-sportová základna". All copy in Czech.

## Colors
- --cz-orange: #E84A1A  (primary accent — CTAs, highlights, active)
- --cz-orange-dark: #B83A12 (hover/pressed)
- --cz-orange-glow: #FF6B35 (secondary highlight)
- --cz-black: #0A0A0A (primary bg)
- --cz-black-mid: #111111 (cards)
- --cz-black-light: #1A1A1A (secondary cards)
- --cz-gray-dark: #2A2A2A (borders/dividers)
- --cz-gray-mid: #555555 (secondary text)
- --cz-gray-light: #888888 (tertiary text)
- --cz-white: #FFFFFF
- --cz-white-soft: #E8E8E8
- --cz-grid-line: rgba(255,255,255,0.06)
Rules: orange is accent ONLY (max 2–3 per layout), never orange-on-orange/white-on-white, never colors outside palette. Bg hierarchy black→black-mid→black-light. WCAG AA.

## Type
- Display/headings: 'Bebas Neue' — ALWAYS uppercase, white on dark only. Hero 72–96px, section 48–64px, subsection 32–40px.
- UI/body: 'Inter' — body 15–16px/1.7 white-soft, never <14px. Card title 20–24px Inter 700.
- Labels/eyebrows/prices-secondary/technical: 'Space Mono' — uppercase, tracking, orange or gray-light, 11–13px.
- Price amount: Bebas Neue large, 36–56px.
Google Fonts: Bebas+Neue, Inter:300;400;500;600;700, Space+Mono:400;700.

## Graphic language
- Signature subtle grid overlay on hero/poster bgs (40px, rgba white .04 lines).
- One thin orange accent line / heading underline / top-border per section, sparingly.
- Optional thin orange corner brackets framing scores/prices/featured.
- Sharp edges: border-radius max 4px (pill badges 100px). No soft drop shadows (sharp 4px 4px 0 orange only). No gradients except subtle orange radial glow on hero (≤0.15 opacity). Typographic/graphic, no stock-photo look.
- 8px base grid; generous padding (≥24px mobile, ≥64px desktop sections).

## Voice
Czech, uppercase Bebas display, direct/confident/no fluff, no exclamation in running text. Prices show "Kč", no decimals unless needed. CTAs = active verbs (REZERVOVAT, HRÁT TEĎ, ZOBRAZIT PROGRAM).
